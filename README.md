# 📖 Overview

This project is a **frontend recreation** of the Qualium website, built as part of a technical skill demonstration.

The primary objective was to recreate the provided user interface using modern frontend technologies while maintaining clean code, responsive layouts, and reusable components.

> **Disclaimer:** This is **not** the original Qualium product. It was recreated solely for learning, portfolio, and technical assessment purposes. The original branding, concept, and intellectual property belong to their respective owners.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Nirisha22/qualium-redesign.git
```

Navigate to the project

```bash
cd qualium-redesign
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
